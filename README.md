## Frequently Asked Questions

### "Is this legal?"

Yes. Cold email is 100% legal in the US (CAN-SPAM Act) and most countries, as long as you:
- Include an unsubscribe link (Instantly.ai does this automatically)
- Don't lie about who you are
- Actually provide value

This is how B2B sales works. If you're targeting businesses, you're good.

### "Will I get blacklisted?"

Not if you follow the system.

The agent is designed to:
- Warm up accounts for 14 days before sending
- Send only 20 emails per account per day (safe limit)
- Rotate across multiple accounts
- Use proper email infrastructure (SPF, DKIM, DMARC)

**This is the SAME system used by agencies sending millions of emails.**

### "Do I need to be technical?"

No.

If you can copy/paste commands, you can set this up.
